# 🎓 Student Management System in C

A simple and beginner-friendly **Student Management System** developed using the **C programming language**. This project helps manage student records efficiently using file handling and structures. It is designed as a first-year programming project to strengthen core C concepts and demonstrate basic CRUD (Create, Read, Update, Delete) operations.

---

## 📌 Features

- ➕ Add New Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student Record
- 💾 Store Data Using File Handling
- 📂 Menu-Driven Console Interface

---
